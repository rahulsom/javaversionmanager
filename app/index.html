jvm.sh

Install java with ease on (almost) any platform.

Demo

Installing

mkdir -p ~/bin && cd ~/bin
curl -s http://javaversionmanager.appspot.com/jvm.sh -o jvm.sh
echo "alias jvm='source ~/bin/jvm.sh'" >> ~/.bashrc # or ~/.zshrc
alias jvm='source ~/bin/jvm.sh'
